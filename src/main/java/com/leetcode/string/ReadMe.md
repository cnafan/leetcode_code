## 常用方法
- split(" ", 3)
- concat()
- string.equal()
- String.valueOf(chars) // 字符数组转字符串
- Arrays.toString // 数组转字符串
- S.toCharArray() // 字符串转数组
- Character.isDigit(char) // 判断一个字符是否为数字
- string.contain(string) // 是否包含字符串
- string.indexof(string) // 查找字串位置
